# Athlotech Landing Page

This is the landing page for Athlotech, an AI-powered mental coaching and precision training analysis platform.

## Features
- Modern, responsive design
- Glassmorphism effects
- Google Sheets integration for Beta Access signups

## Setup
1. Open `index.html` in your browser.
2. The form is connected to a Google Apps Script backend.

## Deployment
This project is ready to be deployed on Vercel or Netlify.
